Tipos de dados: 

Texto: str 
Numérico: int , float , complex 
Sequência:  list, tuple, range
Mapeamento: dict
Definições: set, frozenset
Booleano: bool
Binário: bytes, bytearray, memoryview 

Nenhum tipo de variável tem um valor padrão, deve ser definido inicialmente.

Você pode visualizar o tipo de dado de uma variável usando a função type (): 
print(type(x))
