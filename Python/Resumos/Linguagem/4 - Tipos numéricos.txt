Números 

Há três tipos numéricos em python: 
int
float
complex 

int é um número inteiro, sem decimais e de um tamanho ilimitado.
 
float é um número real, podendo conter ou não decimais. (float também pode conter um 'e', representando uma potencia de 10) 

complex é um número complexo, este contém um 'j' como unidade imaginária. 
formas: bj, a + bj 

É possível converter o tipo de uma variável em outra. Ex: 
x = 1 # int 
a = float (x)
É impossível converter um número complexo em outro tipo. 

Python tem um módulo chamado random que pode ser usado para gerar números aleatórios. 
Uso: 
import random
print(random.randrange(a,b))

Sendo 'a' o valor inicial e o 'b' o valor final do intervalo.
