If else 
Python suporta as seguintes condições lógicas: 

Igualdade: a == b
Diferença: a != b
Menor que: a < b
Menor ou igual a: a <= b
Maior que: a > b
Maior ou igual a: a >= b

Essa estrutura condicional é escrita usando a palavra-chave if 
a = 33 
b = 200 
if b > a: 
   print("B é maior que A") 

Lembrando que indentação é importante já que python não possui fechamento, então o bloco de código que pertence a estrutura é definido pela indentação, retorna erro se não for feita.

A palavra-chave elif é usada no seguinte caso: se as condições anteriores não forem satisfeitas, então tente essa. 
Ex: 
a = 33
b = 33 
if b > a: 
   print("b é maior que a") 
elif a ==b:
   print("a e b são iguais") 

A palavra-chave else é usada para corresponder a qualquer outro caso que não estiver nas condições citadas anteriormente. 
Ex: 
a = 33
b = 33 
if b > a: 
   print("b é maior que a") 
elif a ==b:
   print("a e b são iguais") 
else 
   print("a é maior que b")

É possível ter else sem elif: 
  if b > a: 
    print("b é maior que a") 
 else
   print ("b não é maior que a") 

Se você tiver apenas uma ação para executar, é possível executar o comando em uma linha: 
if a > b: print("a é maior que b") 

É possível fazer o mesmo com if else
print("a é maior que b") if a > b else print("a não é maior que b")

Pode-se fazer isso com outras condicionais:

print ("a é maior que b") if a > b else print("a é igual a b") if a==b else print("b é maior que a")



Operadores Lógicos:

and define que a ação só é executada se todas as condições forem satisfeitas. 
Ex: 
if a > b and c > a: 
   print("Ambas as condições são verdadeiras") 

ou define que a ação é executada se pelo menos uma das condições for satisfeita. 
Ex:
 if a > b or c > a: 
   print("Pelo menos uma das condições são verdadeiras")

É possível encadear multiplos if's: 
if x > 10: 
   print ("É maior do que 10") 
   if x > 20: 
      print ("Também é maior do que 20")
   else 
      print ("Mas não maior que 20.") 

Se por algum motivo haver um if vazio, use a palavra chave pass para prevenir um erro. 
if b > a: 
   pass
