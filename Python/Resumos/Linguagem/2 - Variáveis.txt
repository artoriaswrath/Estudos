Variáveis são declaradas da seguinte forma: 

variavel1 : int
variavel2 : float 

Variáveis do mesmo tipo não podem ser declaradas em sequência:
Errado: variavel1, variavel2 : float 
Certo: variavel1 : float; altura: float 
Recomendado: 
variavel1 : float
variavel2 : float

Valores são atribuídos ás variáveis da seguinte forma: 
variável1= 1 
variável2 = 2
variável3 = "texto"
Declaração de texto deve ser feita com aspas ou aspas duplas.

Declaração do tipo da variável não é obrigatória, essa declaração é feita automaticamente durante a definição de seus valores iniciais.

Variáveis podem mudar seu tipo durante a execução do código, isso pode ser feito com o casting: 
variável = tipo(valor)
Exemplo: 
x = float(1) 
y = int(3) 

Casting não é obrigatório, digitar um valor de outro tipo é o suficiente. 

Variáveis possuem regras de nome para serem criadas: 
   1- O nome de uma variável deve começar com uma letra ou o simbolo underscore (_)
   2- O nome de uma variável não pode começar com um número. 
   3- O nome de uma variável só pode conter caracteres alfanuméricos e underscores (A-z, 0-9 e _) 
   4- Nomes de variáveis diferenciam maiúsculo e minúsculo. (variavel, Variável e VARIÁVEL são três         variáveis diferentes).

Variáveis que são criadas fora de funções são são conhecidas como variáveis globais, elas podem ser usadas tanto dentro quanto fora da função. 
Uma variável criada fora da função com um valor x, e depois definida dentro de uma função com o valor y será usada localmente dentro da função com o valor y, entretanto, fora da função, o valor dela permanece x. 
Variáveis criadas dentro de uma função só podem ser usadas localmente dentro daquela função.

Para criar uma variável global dentro de uma função, usa-se a keyword "global" 
Exemplo: 
função()
   global x 
   x = 10

Python permite que múltiplas variáveis recebam valores em uma linha: 
x, y, z = 1, 2 e 3
Ou que múltiplas variáveis recebam o mesmo valor:
x = y = z = 1
