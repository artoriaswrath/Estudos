Array: list

Ordenado, mutável e indexado. É escrito com colchetes.
Uso: 
A = ["elemento1", "elemento2", "elemento3"] 
print(A) retorna ["elemento1", "elemento2", "elemento3"], ignorando os negativos.
Na prática é criado o seguinte vetor na memória: 
A[0]= "elemento1"
A[1]= "elemento2" 
A[2]= "elemento3" 
A[-1]= "elemento3" 
A[-2]= "elemento2"
Esses itens podem ser acessados da seguinte forma
Ex: 
print(A[0]) retorna "elemento1"
print(A[1:]) retorna "elemento2" pra frente, ou seja, "elemento2" e "elemento3"
print(A[0:1]) retorna "elemento1" 
print(A[0:-1]) retorna ["elemento1", "elemento2"]

Note que o fechamento do intervalo não é incluído. 

É possível alterar o valor de uma lista dessa forma: 
A[1] = "novoelemento"

É possível imprimir na tela todos os elementos dessa list usando o loop (Estrutura repetitiva) "for":
for x in A: 
   print(x)

Para determinar se um ítem especificado pertence a tal lista, usa-se a palavra-chave "in"
A = ["elemento1", "elemento2", "elemento3"] 
if "elemento2" in A: 
   print("Sim, elemento2 está em A") 

Para determinar a quantidade de elementos da lista, use a função len(): 
print(len(A))

 Há várias maneiras de adicionar itens á uma list, para isso usamos métodos de lista: 

append() adiciona um item ao inal da list. 
Uso:
A.append("elemento4")

insert() adiciona um ítem a um índice específico da list
Uso:
A.insert (3, "elemento4")

É possível remover também:

remove () remove um ítem especificado
Uso: 
A.remove ("elemento4") 

pop () remove um índice específico, ou o ultimo ítem se o índice não for especificado. 
Uso:
A.pop()

A palavra-chave del remove o índice especificado: 
Uso: 
del A[0] 
Ou então apagar a lista inteira: 
del A


O método clear() esvazia a lista: 
A.clear() 


Não é possível copiar uma lista na outra assim:
list1 = list2 
Porque se fizer isso, list2 vai apenas ser uma referência ao list1. Todas as mudanças feitas em list1 serão feitas em list2. 

Para fazer uma cópia real e independente, use o metodo copy() 
Ex: 
list2 = list1.copy() 

Ou com o método list()
Ex: 
list2 = list(list1) 

É possível juntar duas listas da seguinte forma: 
list3 = list1 + list2 
A ordem é respeitada. 

Também é possível usar o metodo append()
for x in list2: 
   list1.append(x)

Assim como o metodo extend() 
list1.extend(list2) 

É possível criar uma lista com a função list() 
A = list(("elemento1", "elemento2", "elemento3"))
