Sintaxe 

Python usa a indentação para indicar um bloco de código. 

Exemplo: 
if 5 > 2: 
   print ("Cinco é maior que dois") 

Se essa indentação não for feita, o interpretador retornará um erro. 

O número de espaços  está a critério do usuário, mas deve haver pelo menos um. 

Exemplo: 
if 5 > 2 and 5 > 3:
   print("Cinco é maior que dois") 
   print("Cinco é maior que três")

Deve haver um alinhamento na indentação, o mesmo número de espaços usados deve permanecer até o fim do bloco.

Comentários: 
É possível criar comentários no código através do caractere "#". 
Do "#" até o final da linha, é considerado comentário e não é interpretado pela IDE.
Colocar um # na frente de uma linha de código impede o interpretador de executa-la.
Também é póssível criar um comentário de multiplas linhas utilizando """
Exemplo: 
"""
Esse 
é
um 
Comentário
de
Multiplas 
Linhas 
"""
Você pode fazer isso desde que o texto entre as aspas não estejam associadas com nenhuma variável.
